# Checklist — Story Definition of Done

Après implémentation par le dev-agent :

- [ ] Build passe : `npm run build`
- [ ] TypeScript strict : `npx tsc --noEmit`
- [ ] Lint passe : `npm run lint`
- [ ] Tous les critères d'acceptation de la story sont remplis
- [ ] Pas de `any` TypeScript introduit
- [ ] Pas de `style={{}}` non justifié
- [ ] Pas de fichier > 200 lignes créé (sauf exception documentée)
- [ ] Pas de secret exposé côté client
- [ ] Explication fournie au PO pour chaque fichier modifié
