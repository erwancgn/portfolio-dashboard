# GH-30 — [EPIC 6] Agent Surveillance

## Objectif
Surveiller automatiquement le portfolio et alerter le PO par email si un titre franchit un seuil de baisse, sans que le PO ait à surveiller l'app en permanence.

## User Stories
- [ ] GH-31 / US-014 : Surveillance automatique du portfolio

## Dépendances
- Service email Resend configuré
- Cron Vercel disponible
- Route `/api/quote` opérationnelle

## Critères d'acceptation (epic-level)
- [ ] Les alertes sont envoyées par email via Resend
- [ ] Le cron tourne toutes les 15 min les jours ouvrés
- [ ] Les seuils sont configurables par titre
- [ ] L'historique des alertes est visible dans l'app
