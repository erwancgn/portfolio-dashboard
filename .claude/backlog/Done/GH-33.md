# GH-33 — [US-015] Chat IA avec contexte portfolio

## Contexte
Partie de l'EPIC 7 Agent Chat intégré (GH-32). La route de chat existe déjà (`src/app/api/analyse/chat/route.ts`) et le composant `ChatIA.tsx` est présent. Cette story vise à injecter automatiquement le contexte portfolio et à afficher un compteur de consommation quotidienne.

## Description
En tant que PO, je veux poser des questions sur mon portfolio dans l'app, afin d'obtenir des analyses sans recopier mon contexte manuellement.

## Ce qui doit être fait
- [ ] Modifier `src/app/api/analyse/chat/route.ts` — injecter le contexte portfolio complet en system prompt
- [ ] Ajouter la récupération des positions dans le system prompt (positions, PRU, P&L, allocation)
- [ ] Afficher le compteur de tokens consommés aujourd'hui dans `src/components/analyse/ChatIA.tsx`
- [ ] Ajouter une limite quotidienne configurable (variable d'environnement)
- [ ] Retourner HTTP 429 si la limite quotidienne est atteinte
