# GH-75 — Rapport fiscal — imposition, plus-values et déclaration annuelle

## Contexte
Partie de l'EPIC 10 Fiscalité & reporting (GH-36). Les fichiers de base existent déjà (`src/lib/fiscal.ts`, `src/app/api/fiscal/route.ts`, `src/components/fiscal/FiscalReport.tsx`). Cette story vise à compléter et fiabiliser l'implémentation.

## Description
En tant qu'investisseur français, je veux un rapport fiscal annuel de mon portfolio, pour préparer ma déclaration d'impôts et vérifier que les taxes appliquées sont correctes.

- [ ] Vérifier et compléter `src/app/api/fiscal/route.ts` — filtre par année fiscale
- [ ] Compléter `src/components/fiscal/FiscalReport.tsx` — sections par enveloppe + totaux
- [ ] Ajouter l'export CSV depuis le composant FiscalReport
- [ ] Ajouter le bandeau d'alerte abattement crypto (< 305€)
- [ ] Ajouter la page `/dashboard/fiscal` si non existante

## Critères d'acceptation
- [ ] CA1 : Filtre par année fiscale (2024, 2025…) fonctionnel
- [ ] CA2 : CTO : flat tax 30% calculée sur les plus-values nettes réalisées
- [ ] CA3 : PEA : 0% dans l'enveloppe, 17.2% PS si retrait avant 5 ans
- [ ] CA4 : Crypto : flat tax 30% sur cessions > 305€/an (abattement appliqué)
- [ ] CA5 : Tableau des ventes avec PV brute, taxe calculée, PV nette par titre
- [ ] CA6 : Totaux consolidés : PV totale, impôts payés, net perçu
- [ ] CA7 : Export CSV téléchargeable pour le comptable
- [ ] CA8 : Bandeau d'alerte visible si cessions crypto < 305€ (exonération totale)
