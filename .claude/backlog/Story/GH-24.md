# GH-24 — [US-013] Voir l'historique des passages DCA

## Contexte
Partie de l'EPIC 4 DCA (GH-4). Permet au PO de consulter tous ses passages DCA pour suivre sa discipline d'investissement. Reporté en v1.5.

## Description
En tant que PO, je veux voir l'historique de tous mes passages DCA, afin de suivre ma discipline d'investissement et le montant total investi via DCA.

## Ce qui doit être fait
- [ ] Créer la route `GET /api/dca/history` — lecture table `dca_history` triée par date DESC
- [ ] Créer le composant `src/components/dca/DcaHistoryTable.tsx` — tableau historique
- [ ] Calculer et afficher le total investi via DCA (somme des montants)
- [ ] Intégrer la vue dans la page DCA

- [ ] CA1 : Tableau avec colonnes Date · Ticker · Montant · Prix · Quantité · Nouveau PRU

- [ ] CA3 : Total investi via DCA affiché en bas du tableau
- [ ] CA4 : Les données sont cohérentes avec les positions (même PRU que dans le portfolio)
