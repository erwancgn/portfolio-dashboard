# GH-85 — [US-UX-04] Autocomplete — fermeture immédiate au clic extérieur

## Contexte
Les dropdowns de suggestions (ticker search, nom d'actif) restent affichées trop longtemps et nécessitent plusieurs clics pour disparaître. Le comportement est frustrant.

## Description
En tant qu'utilisateur, je veux que les suggestions disparaissent immédiatement quand je clique ailleurs ou que je sélectionne un élément.

## Ce qui doit être fait
- [ ] Identifier tous les composants avec autocomplete/suggestions (formulaire ajout position, recherche analyse rapide, etc.)
- [ ] Vérifier et corriger le handler `onBlur` / `onClickOutside` (useRef + event listener)
- [ ] À la sélection d'un item : fermer immédiatement le dropdown
- [ ] Au clic en dehors : fermer immédiatement le dropdown
- [ ] Pas de délai artificiel (`setTimeout` inutile)

## Critères d'acceptation
- [ ] CA1 : Un seul clic en dehors ferme le dropdown
- [ ] CA2 : La sélection d'un item ferme le dropdown immédiatement
- [ ] CA3 : Comportement cohérent sur tous les champs avec suggestions

S — correction event handlers sur composants existants
